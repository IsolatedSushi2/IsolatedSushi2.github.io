Annotated LiDaR Visualiser

Description

PandaSet is the result of a collaberation between Hesai's LiDAR sensors with Scale AI's data annotation. This project is an interactive visualiser for this dataset. The data is collected using a specialised car as shown below.

LiDaR Point Cloud

A typical lidar sensor emits pulsed light waves from a laser into the environment. These pulses bounce off surrounding objects and return to the sensor. The sensor uses the time it took for each pulse to return to the sensor to calculate the distance it traveled. Repeating this process millions of times per second creates a precise, real-time 3D map of the environment. This map can be visualised in a big point clound.

Time Stamps

The dataset contains several trips, each consisting of around 80 timestamps. These timestamps are spaced a few seconds apart, meaning the car travels a few hundred meters between them. During recording, the car drives through streets while using its LiDAR sensors and cameras. Its trajectory can be visualized using the colored line.

Colour projection

Besides the lidar sensors, the car also records the images around itself. With this information we can colorize the points in the pointcloud. We do this by projecting the points from world space to image space, and reading the colour from the pixels. By doing this for all the cameras, we can get an accurate colorization of the environment.

Segmentation

The dataset also provides a segmentation. This means that every point in the pointcloud is labeled to be a certain class. For example, classes could be: cars, buildings, road markings et cetera. This is incredibly useful when training classifiers for distinguishing cars from other objects by using supervised machine learning. This segmentation can also be used for creating masks in order to disable the visualisation for some classes entirely.
