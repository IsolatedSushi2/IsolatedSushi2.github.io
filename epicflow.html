Simulating Multi-project Management

Description

Developed a project porfolio simulator in order to interactively simulate and analyse decision outcomes. Used as a training tool to teach multi-resource management concepts in educative settings.

Pipeline

The project visualises the projects using a gantt chart. The projects are represented as a series of tasks, each with a start and end date. The tasks are colour-coded to to indicate which worker can work on them. Various priority mechanisms are implemented to allow the user to observe the effects on the final performance metrics such as profit, worker utilisation and project completion time.

Dashboard

In the dashboard the user can see the state of the projects and performance metrics for all past timesteps.

Graphs

Several graphs are available, for example, showing how worker load, efficiency, output, and capacity change over time to support personnel decisions.

Animated Visualisers

Additional animated visualisers are included to show how scheduled tasks accumulate delays, helping to identify bottlenecks in the project portfolio.
