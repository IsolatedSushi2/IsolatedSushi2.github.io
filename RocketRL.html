Landing Booster Rockets using RL

Description

A simulation environment was created to land a rocket using 4 cold gas thrusters for orientation and thrust vectoring for directional control. Reinforcement Learning was used to train a neural network to land a Falcon 9 rocket on a drone ship. The training was done using the Unity ML-Agents toolkit, which allows for easy integration of Unity environments with Python-based reinforcement learning algorithms. A follow-up project was done to see whether completetly disabling one of its key components, in this case the cold gas thruster places on the right of the rocket, and see if the AI could overcome this limitation.
